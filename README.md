# NexaGPT 🤖

NexaGPT is a full-stack AI assistant that allows users to chat, speak, and learn with an intelligent system.

## 🚀 Features
- 💬 Smart AI chat responses
- 🎤 Voice input (speech to text)
- 🔊 Voice output (text to speech)
- 🧠 Memory system (remembers conversations)
- 📘 Coding tutor mode (teaches step-by-step)
- ⚡ Real-time responses using AI API

## 🛠️ Tech Stack
- Node.js (Backend)
- Express.js
- OpenAI API
- HTML, CSS, JavaScript (Frontend)

## ⚙️ Installation (Local Setup)
1. Clone the repository:
git clone https://github.com/your-username/nexagpt.git

2. Install dependencies:
npm install

3. Create a .env file and add your API key:
OPENAI_API_KEY=your_key_here

4. Run the server:
npm start

5. Open in browser:
http://localhost:3000

## 🌍 Deployment
